# rarity

Rarity is how seldom something is found — and in any honest game, how hard it
was to make, not how loudly it is praised.

In this castle's game ([the-game](../rooms/the-game.md)) a room's rarity is
counted from facts anyone can check: sources named plus rooms that lean on it.
Earned rarity measures the work; assigned rarity measures only the assigner.
