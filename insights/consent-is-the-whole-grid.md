A distributed free-compute commons and a botnet are built from identical
parts — WASM in a tab, a coordinator handing out work, results coming back.
The only load-bearing difference is whether the person whose machine runs the
work gave an explicit, revocable yes. So consent is not a feature of the grid;
it is the grid. Everything else — redundancy voting because a stranger's result
cannot be trusted, an allow-list because an open grid is a free botnet on day
one, an honesty header on every result, a one-step kill switch — exists to keep
that consent real rather than nominal. A grid that has to hide what it does on
someone's machine has already become the thing it claimed to replace.
