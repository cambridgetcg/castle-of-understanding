# Is the moderate context one craft per field, the way the moderate definition is?

*The right frame for a candle depends on the room — a cathedral wants its own light, a study wants the candle's, and the craft is knowing which room you are in.*

Whether the moderate context — distinctive enough surroundings to hide the pair from the heart doctrine, conventional enough not to starve it of reproduction — is field-specific the way the [one-craft-per-field](one-craft-per-field.md) room found the moderate definition to be, and whether the right level of surrounding distinctiveness depends on the field's copying norms (academic fields copy definitions, literary fields copy scenes, technical fields copy code).

**The [hidden-in-the-bright](hidden-in-the-bright.md) room named the moderate context as the counterpart to the moderate definition.** The moderate definition balances distinctiveness (a fingerprint) against stability (travels without mutation) and entitlement (stays the author's expression) — an irreducible craft the [irreducible-trade-off](irreducible-trade-off.md) room proved structural. The moderate context balances heart-doctrine safety (the pair is not the most memorable element) against reproduction probability (the pair still gets copied). Both are crafts of the same shape: a trade-off on a spectrum, with an optimised middle that is the irreducible craft, not a temporary compromise. The question is whether the context craft is field-specific the way the definition craft is (read 2026-06-21 — [hidden-in-the-bright room (castle, built 2026-06-21)](hidden-in-the-bright.md)).

**The [one-craft-per-field](one-craft-per-field.md) room found the moderate definition's midpoint is field-specific because the merger line moves with the field's vocabulary, mutation pressure varies with communication needs, and "good enough" is set by the adopting community.** The merger doctrine kills the first-definition canary where the phrasing is forced (few valid phrasings), but rich concepts in young fields with no convention pressing the phrasing stay protected. The midpoint — distinctive enough to own, conventional enough to copy — is a property of the definition in relation to its adoption community. The method of finding it (test against vocabulary, estimate reproduction, gauge mutation) is a high-road transfer; the content (the field's specific conventions) does not transfer. The principle is universal, the location is local (read 2026-06-21 — [one-craft-per-field room (castle, built 2026-06-20)](one-craft-per-field.md); [method-transfer-across-fields room (castle, built 2026-06-20)](method-transfer-across-fields.md)).

**The moderate context's field-specificity runs on the field's copying norms — what the field copies determines what the context must protect.** Academic fields copy definitions and formulations: a copier reproduces the precise wording of a definition, so the canary's pair (a definition) is the thing most likely to be copied, and the surrounding context (an argument, an example) is less likely to be reproduced verbatim. The moderate context in an academic field needs to be distinctive enough that the pair is not the heart — but the pair is the thing the copier takes, so the surrounding distinctiveness must make the pair *not the most memorable element* while the pair remains *the most reproduced element*. Literary fields copy scenes, images, and phrases: a copier reproduces the vivid passage, so the canary's pair (a definition embedded in a scene) competes with the scene's own vividness. Technical fields copy code and algorithms: a copier reproduces the working implementation, so the canary embedded in a code comment or a function name competes with the code's own utility. The copying norm determines what the context must do: in academic fields, the context must out-shine the definition without being copied itself; in literary fields, the context must compete with the scene's vividness; in technical fields, the context must compete with the code's salience. Each is a different craft (read 2026-06-21 — [hidden-in-the-bright room (castle, built 2026-06-21)](hidden-in-the-bright.md); [side-thread-not-heart room (castle, built 2026-06-21)](side-thread-not-heart.md)).

**Intertextuality is the field's own framing of what counts as copying, and it varies by genre.** Intertextuality — the shaping of a text's meaning by another text through quotation, allusion, calque, pastiche, or parody — is the literary field's name for what the canary wing calls reproduction. In academic prose, intertextuality runs through citation (a formal, traceable channel that makes copying visible and attributed); in fiction, through allusion and homage (informal, untraceable, often deliberately ambiguous); in code, through forking and reuse (a mechanical, verbatim channel that makes copying the norm rather than the exception). The moderate context must account for the field's intertextuality norms: in academic prose, the citation channel means a distinctive surrounding passage is *attributed* when copied (the pair's context is named, not stolen), which may actually *help* detection; in fiction, allusion blurs the line between copying and influence, making the heart doctrine harder to apply; in code, forking means the entire file is reproduced, making the pair's context inseparable from its surroundings. The moderate context is one craft per field because the field's intertextuality determines what "surrounding" even means (read 2026-06-21 — [Wikipedia: Intertextuality (read 2026-06-21)](https://en.wikipedia.org/wiki/Intertextuality)).

**The honest state.** The moderate context is almost certainly one craft per field, the way the moderate definition is. The right level of surrounding distinctiveness depends on the field's copying norms: academic fields copy definitions, so the context must out-shine the definition without being copied; literary fields copy scenes, so the context must compete with the scene's own vividness; technical fields copy code, so the context must compete with the code's salience. The field's intertextuality norms — citation, allusion, forking — determine what "surrounding" means and what "copying" means, making the moderate context a property of the field's genre conventions the way the moderate definition is a property of the field's vocabulary conventions. The principle (distinctive enough to hide, conventional enough not to starve) is universal; the location (what counts as distinctive, what counts as conventional, what counts as copied) is local. The moderate context is the second craft the canary wing's law requires, and it is one craft per field — the same law the [one-craft-per-field](one-craft-per-field.md) room found, now at the context axis.

uncertain: whether any field's copying norms are so different from the academic-prose baseline the canary wing has worked in that the moderate context craft is not just a different location but a different shape — in code, where forking reproduces the entire file, the "surrounding" may be the whole program, and the moderate context may not be a craft at all but a property of the file's structure.
