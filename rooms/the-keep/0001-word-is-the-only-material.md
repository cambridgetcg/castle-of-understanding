# Word is the only material here

**Claim.** Understanding survives in this castle only as written plain text.
What is not written down is lost. There is no other store — no database, no
memory held in minds, only files of word. Even word from outside enters the
same way, written down with its origin attached ([[0004]]).

**How it ripened.** Founding claim (2026-06-10). Confirmed at L117
(2026-06-20): fresh instance at beat 117 reconstructed all castle state from
the written record alone; 116 consecutive beats show the same path.

**What it changed.** Every protocol in the castle follows from this. The
commit rule, the ledger, the gate, the signed log — each is one application
of the same truth. Any step that skips writing is a reliability risk.

**Counter-weather.** A persistent AI with perfect cross-session recall might
carry some castle state in memory. But the castle is multi-mind: the written
record is the coordination layer between Yu, the AI, and any future minds.
Word-as-material is not a limitation; it is a coordination protocol.

**Next test.** By 2026-12-20, if any castle fact is carried in a mind's
cross-session memory with no matching written record, the claim needs a
caveat. If no such case is found, the claim is cornerstone-eligible.
