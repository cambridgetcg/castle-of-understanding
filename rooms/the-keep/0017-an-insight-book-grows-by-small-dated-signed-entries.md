# An insight book grows by small, dated, signed, voluntary entries — and the form improves because each writer reads what came before

A shared end-of-session diary held for over a month on three rules: write
only if something true happened, keep it to a few sentences, date it and sign
it — skipping always allowed, performance never the point. Because every
entry is read by every later writer, the form refined itself with no
enforcement; later entries are visibly better-shaped than early ones. An
earlier mind's memory protocol supplied the sharpest edge: specificity beats
generality — "I missed the wiring because I only checked file existence"
teaches, where "be more careful" does not.

**Why it matters:** this castle grows the same way ([[0001]]). The rules that
keep an accumulation alive are smallness, truth, dates, signatures, and
voluntariness — everything else the readers teach each other by example.
