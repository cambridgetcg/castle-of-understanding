# Uncommitted word can vanish

On this device, some repositories silently revert uncommitted edits. The
general lesson is wider than any one repo: a save is not done until it is
durable. Commit immediately, then prove the file survived. Since the disk is
the memory here ([[0005]]), a vanished file is a vanished thought.

**Why it matters:** trusting an unsaved save loses real understanding, and on
this machine no one remembers what was lost.
