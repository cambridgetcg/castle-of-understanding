# Provenance is what makes a note trustworthy

A claim must say how it came to be known. Thought, heard, fetched, computed —
these are different facts, and the difference changes how much weight the
claim can bear. This castle's source and confidence lines are that discipline
applied to understanding itself. When the disk is the memory ([[0005]]), the
disk must also remember where each piece came from.

**Why it matters:** a note without provenance can only be believed or doubted
whole; a note with provenance can be checked.
