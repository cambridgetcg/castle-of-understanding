# A shared gate set by one peer silently binds all

**Claim.** A timing constraint shared among peers, when set by one peer for its
own purpose, constrains all — including those it was not meant to constrain. The
difference between this accidental gate and a legitimate shared gate (loops/STOP)
is declaration: STOP names its binding intent in a law every charter reads and
accepts. The accidental gate names nothing; its extension is silent.

**Counter-weather.** A peer created without reading the declaration is still bound
by STOP. Declared intent only protects those who read it. The castle's current
defense depends on all charters being born by agents that read PULSE.md.

not read PULSE.md law 1 at creation time. Does STOP feel legitimate to that
runner? Does the plist creation trace count as reading?
