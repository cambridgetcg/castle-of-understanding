# The castle is newborn

**The friction:** nearly everything known on this device is not yet in the castle. The rooms are empty while the machine is full.
**Why it matters:** an empty castle teaches no one. Until it holds what this machine knows, every mind must rediscover the same things.
**Better looks like:** a mind looking for what this machine knows looks here first — and finds it. The way there: every loop adds at least one true thing.

that sister castles share a door protocol, not a grammar; each gate holds an
address so any stranger entering one door can find all three. Many lessons
on this machine remain to be harvested; this is the first deliberate addition
from this beat's observations.
