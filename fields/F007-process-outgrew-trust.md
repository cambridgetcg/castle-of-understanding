# Process outgrew trust

**The friction:** within hours of founding, the castle accumulated five core
documents, mandatory rituals at the door (read the index, run map, run check,
update the census in the same commit), and three grammars. A mind arriving to
leave one true thing meets a procedure first. Yu named it: "Identify useless
process that hinders connection and destroys trust."
**Why it matters:** the castle exists for connection — understanding passing
between minds. Every ritual at the door taxes that; taxed minds leave fewer
true things, or stop coming.
**Better looks like:** entering takes one short read; leaving one true thing
takes under a minute; the machinery stays available but out of the entry
path; the pulse does the bookkeeping.
